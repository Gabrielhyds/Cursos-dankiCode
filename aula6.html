Os comandos básicos (status, add, commit, log e diff) git status - oq foi modificado git diff - diferença dos arquivos git -help - todos os comandos do git git add . - adiciona todos os arquivos git restore --staged . - remove todos os arquivos já adicionados git commit -m 'arquivos atualizados' - nomeia os arquivos para enviar pro git git log - mostra todas as alterações
